# QrESETChangerz
Python script for converting QR codes from ESET, compiled to exe.

- Graphical tool for transforming  a provisioning ESET security for android QR code, so that its download location is version 5.1.2.0, and not the latest one
-	Suitable for Android 8, which is not supported by newer versions of ESET
- Written in Python and pythonnet library, pyzbar module for decoding QR code, PIL module for image, and others. Compiled with auto-py-to-exe
- A little bit AI assisted in the .NET part, but not AI generated!
